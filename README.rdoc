== espeak

espeak is a speech synthesizer software. It can be used as command-line program, or through shared libraries. This code uses espeak as command-line program. 
For more info about espeak, visit espeak.sourceforge.net.

== requirements

* linux (I couldn't make espeak run on mac os x)
* espeak
* lame
